# Мониторинг событий ядра

## Файлы

| Файл | Описание |
|---|---|
| etw_monitor.cpp / etw_monitor.h | Сессия трассировки ETW, обработка событий процессов и загрузки библиотек, обнаружение инъекций |

## Классы и функции

### EtwMonitor (etw_monitor.h / etw_monitor.cpp)

| Метод                                                               | Описание                                                                             |
| ------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| EtwMonitor()                                                        | Уникальное имя сессии (с PID), список "безопасных" каталогов из переменных окружения |
| ~EtwMonitor()                                                       | Вызов stop()                                                                         |
| stop()                                                              | Остановка трассировки, ожидание потока (до 3 сек, иначе отсоединение)                |
| isRunning() const                                                   | Флаг running_                                                                        |
| setProcessCallback(cb)                                              | Регистрация callback событий запуска/остановки процессов                             |
| getAndClearInjectionAlerts()                                        | Список подозрений на инъекцию с очисткой                                             |
| getStatus() const                                                   | Текстовый статус (active, failed_no_admin, failed_start и другие)                    |
| enableProvider()                                                    | Подписка на системный провайдер процессов с маской 0x30                              |
| openConsumer()                                                      | Открытие потребителя трассировки в реальном времени                                  |
| traceLoop()                                                         | Фоновый поток: блокировка в ProcessTrace, пока сессия активна                        |
| cleanup()                                                           | Закрытие потребителя, остановка сессии                                               |
| eventRecordCallback(pEvent) (статический)                           | Фильтр по GUID провайдера и коду события (1/2/5)                                     |
| handleProcessStart(pEvent) / handleProcessStop(pEvent)              | Извлечение PID/PPID/пути/командной строки, вызов process callback_                   |
| handleImageLoad(pEvent)                                             | Путь загруженного образа; для .dll - analyzeDllLoad                                  |
| isSuspiciousDll(dllPath) const                                      | Не подозрительна, если путь в "безопасном" каталоге или \device\                     |
| isDllHijacking(dllName, dllDir) const                               | true, если имя совпадает с системной библиотекой, но каталог не "безопасный"         |
| getProcessNameByPid(pid) const                                      | Имя файла процесса без пути                                                          |
| getEventStringProperty/getEventUInt32Property(pEvent, propertyName) | Разбор свойства ETW-события через TDH API                                            |
| wstringToUtf8(wstr) const                                           | Преобразование "широкой" строки в UTF-8                                              |
| getCurrentTime() const                                              | Текущее локальное время                                                              |

**start()** - остановка возможной "осиротевшей" сессии с тем же именем, затем создание сессии, включение провайдера, открытие потребителя; при неудаче - откат (cleanup) и переключение вызывающего кода на резервный режим опроса.

**createSession()** - ERROR_ACCESS_DENIED фиксируется как failed_no_admin; при уже существующей сессии - остановка старой и повторная попытка.

**analyzeDllLoad(pid, dllPath)** - загрузка из временной папки (+40), из пользовательских каталогов (+35), подмена системной библиотеки (+60); предупреждение регистрируется при счёте от 35, список ограничен 200 записями.
