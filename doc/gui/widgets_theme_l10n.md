## Переиспользуемые виджеты

Компоненты, используемые более чем на одном экране - вынесены отдельно от screens.

| Файл | Назначение |

| app_title_bar.dart | Заголовок окна приложения (кастомное оформление окна Windows) |

| count_badge.dart | Значок-счётчик |

### Классы и функции - переиспользуемые виджеты

Большинство компонентов - тривиальные StatelessWidget с одним build(). Ниже - только те, где есть собственная логика.

**AppButton** - три варианта оформления (primary/secondary/ghost), build() - switch по типу с веткой по наличию иконки.

**AppCard** - InkWell при заданном onTap, иначе содержимое напрямую.

**AppRoutes.fade(page)** - PageRouteBuilder с фейд-переходом (200/150 мс); большинство экранов строят идентичный переход инлайн, а не вызывают эту фабрику.

**AppTitleBarScaffold** / **AppTitleBar** - кастомный заголовок окна (window_manager); первый добавляет кнопку "назад".

**WindowControlsState** - подписка на WindowListener, синхронизация иконки развернуть/восстановить с реальным состоянием окна.

**BottomSheetShell** - обёртка bottom sheet (закруглённый верх, "ручка", опциональный заголовок); переиспользуется в нескольких листах.

**ConfirmDialog.show(...)** - AlertDialog отмена/подтвердить, опасный цвет при isDestructive: true; возвращает false вместо null при закрытии без выбора.

**NudgeSheet**:

| Метод                  | Описание                                                                           |
| ---------------------- | ---------------------------------------------------------------------------------- |
| show({...})            | Открытие bottom sheet                                                              |
| build(context)         | Заголовок, бейдж файла, статус USB-сканирования, контекст, совет, чек-лист, кнопки |

**resolveKey()** - сопоставление строковых ключей (по 7 на категорию нуджа) с геттерами AppLocalizations; при отсутствии совпадения - возврат ключа как есть.

**ActionButtons** - кнопка сканирования USB блокируется на время сканирования, меняет подпись при повторном сканировании; для категорий безопасности - кнопка "в карантин". Закрытие листа всегда вызывает onEngaged/onIgnored.

**UsbScanStatus** - switch с гвардом по state.status/isClean; _buildError() - единственный захардкоженный русский текст без l10n.

**SectionHeader** - два стиля (iconUppercase/compact) через enum.

**GenericSparkline** / **GenericSparklinePainter** - нормализация точек в [minValue, maxValue] с защитой от деления на ноль (плоская линия по центру при range == 0), заливка и линия, опциональная акцентная точка.

**StatChip** - плитка иконка+число+подпись, тривиален.

## Тема (theme/)

| Файл | Назначение |
|---|---|
| app_theme.dart | Светлая и тёмная тема приложения |
| spacing.dart | Единые токены отступов |

### Классы и функции - тема

**AppColors** / **AppColorsDark** - статические палитры без логики.

**AdaptiveColors** - конструктор bool isDark, каждый геттер цвета - тернарный выбор между AppColorsDark/AppColors; центральная точка темы, экраны обращаются только к ней.

severityColor(level) - пороги ≥9/≥7/≥4; четвёртая независимая шкала опасности в GUI (после локальных шкал в ThreatCard/ReportCard/ IncidentCard), при этом нигде не вызывается.

**AppTextStyles** - именованные размеры шрифта и готовые TextStyle, не зависят от темы.

**AppTheme.light** / **AppTheme.dark** - конфигурации ThemeData для MaterialApp, используются один раз в main.dart.

**Spacing** - именованные отступы (xs=4 ... xxl=32) и готовые EdgeInsets; соблюдение проверяет гейт tools/check_design.dart.

## Локализация (l10n/)

| Файл | Назначение |
|---|---|
| app_localizations.g.dart | Сгенерированный базовый класс локализации |
| app_localizations_ru.g.dart | Строки на русском |
| app_localizations_en.g.dart | Строки на английском |
| vuln_l10n.dart | Дополнительная локализация для экрана уязвимостей |

Файлы .g.dart генерируются из .arb-файлов, вручную не редактируются.

### Классы и функции - локализация

**AppLocalizations** - абстрактный базовый класс: of(context) через Localizations.of, delegate для MaterialApp; геттер/метод на каждую строку интерфейса; ru/en - реализации.

**getVulnLocalized(id, locale)** - тексты уязвимостей генерируются ядром на русском; функция хранит только английские версии (en), возвращает null для остальных локалей - сигнал использовать текст из ядра как есть.

### Классы и функции - утилиты

**Snack** - единственная точка показа snackbar во всём GUI:

| Метод                                   | Описание                                          |
| --------------------------------------- | ------------------------------------------------- |
| error(context, message, {duration})     | Snackbar с фоном colors.danger                    |
| success(context, message, {duration})   | Snackbar с фоном colors.success                   |
| info(context, message, {duration})      | Snackbar с фоном colors.primary                   |
| _bg(context, kind)                      | AdaptiveColors из AppStateProvider, если доступен |

**show()** - ScaffoldMessenger.maybeOf(context) (не падает без Scaffold); clearSnackBars() перед показом нового - без накопления очереди; длительность по умолчанию 2 секунды.
