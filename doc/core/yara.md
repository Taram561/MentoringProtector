# YARA-движок

## Файлы

| Файл | Описание |
|---|---|
| yara_scanner.cpp / yara_scanner.h | Обёртка над динамически загружаемой библиотекой YARA: компиляция правил, сканирование, разбор совпадений; правила - в data/yara_rules/ |

## Классы и функции

### YaraScanner (yara_scanner.h / yara_scanner.cpp)

| Метод                                                                 | Описание                                                               |
| --------------------------------------------------------------------- | ---------------------------------------------------------------------- |
| YaraScanner() / ~YaraScanner()                                        | Деструктор вызывает shutdown()                                         |
| scan(file_path) (реализация IScanEngine)                              | Вызов scanFile, упаковка в общий ScanEngineResult                      |
| isAvailable() const                                                   | Флаг available_                                                        |
| getRulesCount() const                                                 | Число скомпилированных правил                                          |
| reloadRules(rules_dir)                                                | Освобождение текущих правил, загрузка и компиляция заново              |
| shutdown()                                                            | Освобождение правил, yr_finalize, выгрузка yara.dll                    |
| loadCompiledRules(dir_path)                                           | Загрузка готового бинарного кэша compiled_rules.yrc; 0, если файла нет |
| severityToScore(severity) (статический)                               | Текстовая степень серьёзности правила в очки риска: 80/50/25           |

**initialize(rules_dir)** - однократно: загрузка yara.dll (без неё движок недоступен, но не считается ошибкой) -> yr_initialize() -> попытка загрузить скомпилированный кэш -> при его отсутствии компиляция .yar-файлов заново.

**loadLibrary()** - поиск yara.dll рядом с MentoringProtector_core.dll, в data\, либо до 8 уровней вверх по дереву каталогов; после загрузки - разрешение функций YARA C API через GetProcAddress.

**compileRulesFromDir(dir_path)** - компиляция всех .yar-файлов каталога и подкаталога community/ в единый набор правил, каждому файлу - своё пространство имён; результат по возможности сохраняется как compiled_rules.yrc для быстрой загрузки в следующий раз.

**scanFile(file_path)** - одноразовый объект сканирования поверх скомпилированных правил, таймаут 10 секунд, callback scanCallback; угроза - если список совпадений не пуст, имя - правило с максимальным счётом риска (worst_rule).

**scanCallback(...)** (статический) - извлечение имени правила, тегов и метаданных из совместимой по памяти структуры YR_RULE_COMPAT, с проверкой каждой строки на печатаемый ASCII (safeAsciiStr); счёт риска суммируется в общий счёт файла.
