# Карантин

| Файл | Описание |
|---|---|
| quarantine.cpp / quarantine.h | Перемещение в карантин, шифрование, восстановление, база записей карантина |

## Классы и функции

### QuarantineManager (quarantine.h / quarantine.cpp)

| Метод                                                                             | Описание                                                                                                        |
| --------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| QuarantineManager() / QuarantineManager(quarantine_dir)                           | Создание каталога карантина, загрузка/создание мастер-ключа, загрузка базы записей, удаление устаревших записей |
| ~QuarantineManager()                                                              | Сохранение базы, обнуление мастер-ключа в памяти                                                                |
| restoreFile(entry_id)                                                             | Поиск записи по id, вызов restoreFileTo с исходным путём                                                        |
| restoreFileTo(entry_id, dest_path)                                                | Расшифровка файла карантина в указанный путь, удаление зашифрованного файла и записи при успехе                 |
| deleteFile(entry_id)                                                              | Безвозвратное удаление зашифрованного файла и записи                                                            |
| clearAll()                                                                        | Удаление всех зашифрованных файлов, очистка базы                                                                |
| getAllEntries() const                                                             | Копия всего списка записей                                                                                      |
| findById(entry_id) const                                                          | Линейный поиск записи по идентификатору                                                                         |
| getCount() const / getTotalSize() const                                           | Число записей и суммарный размер оригиналов                                                                     |
| generateId() const                                                                | Новый GUID как строка - идентификатор записи                                                                    |
| getCurrentDateTime() const                                                        | Текущее локальное время                                                                                         |
| removeEntry(entry_id)                                                             | Удаление записи из списка по id, без удаления файла                                                             |
| entryToJson(entry) const / parseEntry(json) const                                 | Ручная сериализация/разбор записи в/из JSON                                                                     |
| extractField/extractLongLong/extractInt(json, key) const                          | Посимвольный разбор одного поля JSON по ключу                                                                   |

**quarantineFile(entry)** - защита от TOCTOU: открытие файла с GENERIC_READ | DELETE, отклонение reparse point; удаление оригинала после шифрования через SetFileInformationByHandle/FileDispositionInfo на уже открытом дескрипторе - без отдельного окна гонки между шифрованием и удалением.

**encryptFileByHandle(...) / decryptFile(...)** - AES-256-GCM (аутентифицированное шифрование - обнаруживает изменение содержимого после шифрования) через Windows CNG (BCrypt*), а не устаревший CryptoAPI; буфер расшифрованных данных обнуляется (SecureZeroMemory) даже в путях ошибки; файлы крупнее 512 МБ не шифруются.

**loadOrCreateMasterKey()** - расшифровка существующего ключа через DPAPI; при отсутствии/повреждении - генерация нового случайного ключа.

**saveMasterKey() const** - шифрование через DPAPI, атомарная запись через временный файл и MoveFileExW.

**wipeLegacyEntries()** - записи без магической метки MP03 (старый формат шифрования) удаляются вместе с файлом.

**loadDatabase() / saveDatabase() const** - собственный минималистичный JSON без внешней библиотеки; запись с отсутствующим файлом помечается is_orphan.
