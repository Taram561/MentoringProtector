# Журналирование

## Файлы

| Файл | Описание |
|---|---|
| logger.cpp / logger.h | Потокобезопасная запись сообщений в файл с фоновым сбросом, ротацией и ограничением прав доступа |

## Классы и функции

### Logger (logger.h / logger.cpp)

| Метод | Описание |
|---|---|
| instance() (статический) | Единственный экземпляр; при первом обращении создаёт logs/, применяет ограничения доступа, запускает фоновый сброс |
| log(level, module, message) | Запись строки [время] [уровень] [модуль] сообщение; для Warning/Error - немедленный сброс на диск, каждые 100 записей - проверка ротации |
| debug/info/warning/error(module, msg) | Обёртки над log с фиксированным уровнем |
| backgroundFlush() | Фоновый поток: сброс буфера на диск раз в секунду |
| rotateIfNeeded(logPath) const | При размере файла от 5 МБ - сдвиг истории (.log -> .log.1 -> .log.2) |
| reopenFile() | Закрытие и переоткрытие файла журнала после ротации |
| applyFileDacl(filePath) const / applyDirDacl(dirPath) const | Список контроля доступа (DACL) через SDDL - доступ только владельцу и системе |
| getCurrentTime() const / levelToString(level) const | Форматирование времени и уровня серьёзности |

**Синхронность записи и сброса** - строка попадает в буфер потока синхронно под мьютексом; асинхронен только принудительный сброс буфера на диск: немедленно для Warning/Error, раз в секунду фоновым потоком для остальных уровней.
