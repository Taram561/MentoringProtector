# Обучающие подсказки

## Файлы

| Файл | Описание |
|---|---|
| nudge.h | Базовая структура подсказки и её категории |
| inudge_sink.h | Интерфейс приёмника подсказок |
| nudge_engine.cpp / nudge_engine.h | Приём, дедупликация и очередь подсказок |
| motw_reader.cpp / motw_reader.h | Определение, что файл загружен из интернета (Mark of the Web) |
| script_guard.cpp / script_guard.h | Поиск подозрительных конструкций в скриптовых файлах |
| tray_notifier.cpp / tray_notifier.h | Всплывающее уведомление в системном трее Windows |
| usb_monitor.cpp / usb_monitor.h | Обнаружение подключения съёмных накопителей |

## Классы и функции

### NudgeEngine (nudge_engine.h / nudge_engine.cpp)

| Метод                               | Описание                                    |
| ----------------------------------- | ------------------------------------------- |
| getAndClear()                       | Очередь подсказок под мьютексом, с очисткой |
| makeKey(n) const                    | Ключ дедупликации: код категории + деталь   |
| categoryToId(c) const               | NudgeCategory в числовой идентификатор      |
| getCurrentTime() const              | Текущее локальное время                     |

**emit(nudge)** - для категории "загруженный контейнер" - отдельный час охлаждения на всю категорию; общая дедупликация по ключу "категория + деталь" в окне 10 секунд (контекст дополняется в существующую запись, если у неё его не было); очередь ограничена 100 записями, вытесняются самые старые.

### INudgeSink (inudge_sink.h) и Nudge/NudgeCategory (nudge.h)

Интерфейс из одного метода emit(nudge); Nudge - плоская структура (категория, деталь, контекст, серьёзность, время); isSecurity() const относит к предупреждениям безопасности три из пяти категорий.

### motw (motw_reader.h / motw_reader.cpp)

**isFromInternet(path)** - чтение альтернативного потока данных Zone.Identifier (NTFS ADS); поиск ZoneId= со значением 3+; отсутствие потока трактуется как "не из интернета".

### script_guard (script_guard.h / script_guard.cpp)

**analyze(path_utf8)** - применяется к .ps1/.vbs/.hta/.bat не крупнее 10 МБ, читает первый 1 МБ; поиск 10 характерных токенов (Invoke-Expression, DownloadString, -EncodedCommand, -WindowStyle Hidden, Net.WebClient и другие); подозрительно - при совпадении не менее двух разных токенов.

### TrayNotifier (tray_notifier.h / tray_notifier.cpp)

| Метод                      | Описание                                                                   |
| -------------------------- | -------------------------------------------------------------------------- |
| stop()                     | Сообщение остановки скрытому окну, ожидание потока                         |
| showBalloon(title, text)   | Сохранение текста, запрос показа скрытому окну                             |
| consumeClick() const       | true и сброс флага, если был клик с момента последней проверки             |
| threadMain(readyEvent)     | Фоновый поток со скрытым окном сообщений, регистрация значка в трее        |
| wndProc(...) (статический) | Обработка клика по значку, запроса показать уведомление, запроса остановки |

**start()** - создание потока со скрытым окном, ожидание сигнала готовности (до 5 секунд) перед возвратом управления.

### UsbMonitor (usb_monitor.h / usb_monitor.cpp)

| Метод                               | Описание                                                           |
| ----------------------------------- | ------------------------------------------------------------------ |
| start(sink)                         | Скрытое окно в отдельном потоке, ожидание готовности (до 3 секунд) |
| stop()                              | Остановка потока, уничтожение окна                                 |
| threadMain(readyEvent)              | Невидимое окно для получения системных сообщений об устройствах    |
| driveLetter(unitMask) (статический) | Буква диска из битовой маски DBT_DEVICEARRIVAL                     |

**wndProc(...)** - реакция на WM_DEVICECHANGE только для съёмных накопителей (DRIVE_REMOVABLE); дедупликация повторных подключений одной буквы диска.
