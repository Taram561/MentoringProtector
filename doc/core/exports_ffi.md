# FFI

## Файлы

| Файл                        | Описание                                                                                                                |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| exports.cpp / exports.h     | Полный список экспортируемых из DLL функций (MP_API), сгруппированных по подсистемам                                    |
| service_ipc_client.h / .cpp | Проверка активности фоновой службы, отправка ей команд через именованный канал                                          |
| json_utils.h                | Экранирование/разбор строк JSON, используется всеми подсистемами ядра                                                   |
| unicode_utils.h             | Преобразование UTF-8 в "широкие" (UTF-16) строки и обратно для вызовов Windows API                                      |
| pch.h / pch.cpp             | Предкомпилированный заголовок сборки (общие системные подключения)                                                      |
| framework.h                 | Минимальное подключение <windows.h> с настройками сборки                                                                |
| main_test.cpp               | Отдельная небольшая точка входа для ручной проверки ключевых функций ядра из консоли, отдельно от автоматических тестов |

## Функции

| Функция                    | Описание                                                 |
| -------------------------- | -------------------------------------------------------- |
| getDllDir() / getDllDirW() | Вычисляют и кэшируют каталог, в котором лежит DLL        |
| resultToJson(result)       | Преобразует ScanResult целиком в JSON                    |
| stringToCharPtr(str)       | Выделяет новый char[] и копирует туда строку             |
| getQuarantine()            | Создает экземпляр QuarantineManager при первом обращении |
| nudgeCategoryName(c)       | Переводит NudgeCategory в строковый идентификатор JSON   |

**initializeCore()** - создание Scanner, загрузка в него базу сигнатур, угроз и эвристические правила, подключает к нему эвристику, сканер архивов и исключения; создает ProcessAnalyzer и RealtimeMonitor; запускает мониторинг USB-устройств.

**ensureYaraInitialized()** - инициализация YARA.

**DllMain(...)** - последовательно останавливает все фоновые потоки и сохраняет несохраненную статистику.

**mp_verify_helper_exe(path)** - проверка подлинности вспомогающего процесса

**start_realtime_monitor()** - проверяет isServiceHosting() - если фоновая служба уже хостит защиту в реальном времени, то сразу возвращает 1 с пометкой hosted_by_service: true, не запуская второй набор ReadDirectoryChangesW-потоков.
