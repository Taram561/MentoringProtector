# База сигнатур

## Файлы

| Файл | Описание |
|---|---|
| signatures.cpp / signatures.h | Загрузка и поиск по базе сигнатур; база - data/signatures.msdb, около 540 тысяч записей |

## Классы и функции

### SignatureDatabase (signatures.h / signatures.cpp)

| Метод                                      | Описание                                            |
| ------------------------------------------ | --------------------------------------------------- |
| SignatureDatabase() / ~SignatureDatabase() | Тривиальны                                          |
| findByHash(hash) const                     | Поиск записи по точному значению хэша в хэш-таблице |
| getCount() const                           | Число загруженных записей                           |
| isLoaded() const                           | true, если загрузка дала хотя бы одну запись        |

**loadFromFile(db_path)** - построчное чтение, комментарии (#) и неразбираемые строки пропускаются; записи угроз для Android (Andr./Android.) намеренно отбрасываются - продукт защищает только Windows.

**parseLine(line, record)** - формат хэш:размер:название по первым двум : (название может содержать двоеточия); ошибка разбора размера не проваливает запись - размер выставляется в -1.
